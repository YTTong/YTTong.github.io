技术面+HR面+牛客网面经总结

工作职责

在软件项目经理的领导下，配合完成程序设计和开发；
按产品需求进行软件设计和编码实现，确保安全、质量和性能；
参与内部测试、部署、实施等工作；
分析并解决软件开发过程中的问题。

任职资格

本科及以上学历，计算机类相关专业；
对面向对象程序设计和开发有较深入理解，熟悉常用设计模式；
具备较为扎实的Java语言开发功底，熟悉java EE平台相关知识；
熟悉Struts/SpringMVC/Hibernate等开发框架；
熟悉Tomcat、JBoss、Weblogic等应用服务器中的一种。

我的面经

一面（9.19面试，30-40分钟）

二面（20分钟）

问项目中遇到什么问题，后来怎么解决的，给你带来了什么，对后面的生活学习有没有帮助，这中间又是各种被怼，各种反问我一些问题，已经记不太清了。如果这个项目重新让你做一次你会考虑怎么做，你觉得最关键的决定项目能否做成的因素都有什么？最后问我有什么问题，聊得还行。

牛客网面经总结：

（一）

  1. 自我介绍

  2. 开始问我实习的单位的情况。就是一个水水的有点国企性质的软件公司（学校安排的实训）。问我做了什么，其实我真没做什么，就胡乱说了一些，扯到了 Spring Boot。

Spring

  1. 开始问 Spring Boot 和 Spring 的区别，你觉得 Spring Boot 哪里好。

网络

  1. 访问一个网站，比如 www.baidu.com 的时候，是怎么个流程。

  2. HTTP 1.0/1.1 的区别。长连接是怎么实现的。

数据结构

  1. 有哪些数据结构

  2. HashMap 的数据结构是什么。

  3. 有哪些树；B、B+、红黑、AVL 树的区别。

  4. LinkedList 的数据结构是什么；逆序输出它的时间空间复杂度多少。

    链表，时间复杂度：log(n)，空间复杂度：

  5. 索引的数据结构是什么；为什么用 B+ 树，不用其他的树，比如红黑、二叉树、AVL 树。

Java

  1. 有哪些创建线程的方式。

  2. 三个线程，A、B同时 start，要求 C 在 A、B 运行完之后才运行，求解决方案。

  3. 线程通信的方式。

  4. JVM 的结构模型是什么。

  5. 哪些是线程共享的。

    栈和程序计数器是私有的

  6. 内存分为哪几块（指的是新生代这种）

二面（HR面，20min）

没有自我介绍

为什么想要从事 Java 开发。

来海康的原因。

一件让你有成就感的事情。

将来的规划。

大学期间的是否有突出的经历。

你怎么看过往的大学。

怎么评价自己，别人怎么评价自己。

兴趣爱好。

有哪些 offer。

期望薪资是多少。

有什么问我的。问：啥时候有接下来的通知。答：大概 20 号多点关注官网出通知。

（二）

项目架构

redis和mysql同步，redis数据失效怎么办，数据更新怎么办。

讲一下堆排序

讲一下AVL树，讲一下B,B plus(打不出来加号╯^╰) ,AVL,RBTree区别于联系。

死锁，什么情况会发生死锁，遇见死锁有什么解决方法。

spring 声明式事务如果实现。

spring 管理的controller是单例的，怎么实现的？

分布式情况下controller是怎么处理多线程不安全的？

linux如果查看一个进程的状态。

如果我的Java进程挂了，linux下怎么处理？

有什么想问的？

（三）

JVM说一下吧，越详细越好。内存划分、GC什么的。分代收集？

String s = “Hello World”，涉及到内存的那些位置？String类型是Java的基本数据类型么？它可以被继承么？

StringBuilder和StringBuffer的区别？

sleep和wait的区别？会不会释放锁？

Java中一个线程可以如何实现？

Java线程池怎么实现？分类？

如果说要给你调整工作岗位你可以接受么？

你还有什么要问我的？

（四）

sql，写一个查询语句

SQL注入

写几个你熟悉的，除了单例和工厂模式 （代理模式，责任链模式，还有观察值模式，原形模式，桥接模式 ）

（五）

技术面

先自我介绍

讲jvm，讲多线程

他问我有什么想问的，我就说海康做的业务场景有哪些，技术栈有哪些。

HR面

聊经历聊规划聊女朋友和留杭计划什么的

还有对加班和出差接不接受，那肯定是全部接受了

期望薪资口嗨说了个15-20k，hr说可能一进来给不到，我说没关系我更看重海康这个平台。
